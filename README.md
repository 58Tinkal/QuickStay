# QuickStay

QuickStay is a hotel booking web application that allows users to search for hotels, view details, and reserve rooms, while providing admins with tools to manage listings and bookings.[attached_file:1]

## Features

- User registration and login for guests and admins.[attached_file:1]
- Browse a list of available hotels and rooms.
- View hotel details such as photos, descriptions, pricing, location, and amenities.
- Search and filter hotels by city, dates, price range, and other criteria.
- Create and manage room bookings.
- Admin dashboard to manage hotels, rooms, and reservations.
- Responsive UI for desktop and mobile devices.
